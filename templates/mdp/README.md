# MDP Templates

This repository ships a generic GROMACS 2025 explicit-water protein MD template set:

- `verified/gromacs_2025_explicit_water_10ns/ions.mdp`
- `verified/gromacs_2025_explicit_water_10ns/em.mdp`
- `verified/gromacs_2025_explicit_water_10ns/nvt.mdp`
- `verified/gromacs_2025_explicit_water_10ns/npt.mdp`
- `verified/gromacs_2025_explicit_water_10ns/md.mdp`

Review all parameters before production use. Do not treat these as universal scientific defaults.

Historical local templates and third-party CHARMM-GUI example files are not redistributed in this public release.
