# Self-Evolving Skill Rules

The skill can maintain a reusable memory system, but public memory must remain generic.

- Verified fixes can be reused.
- Unsafe fixes require manual approval.
- Project-specific facts must not be promoted to public defaults.
- Scripts created from one case must be generalized and tested before release.
- Never silently change scientific inputs, topology, force fields, ligand chemistry, membrane composition, ion definitions, or trajectory-analysis definitions.
