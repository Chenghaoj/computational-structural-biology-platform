# License Recommendation

Recommended GitHub license: **MIT License**.

Rationale:

- Short, permissive, and familiar for open-source workflow scripts and documentation.
- Easy for academic users to understand and reuse.
- Compatible with keeping scientific responsibility and validation with downstream users.

A `LICENSE` file with MIT terms is included. Before publishing, confirm that all bundled templates and examples are authored by you or otherwise compatible with MIT redistribution. Third-party generated examples from tools such as CHARMM-GUI are intentionally not bundled in this public release.
